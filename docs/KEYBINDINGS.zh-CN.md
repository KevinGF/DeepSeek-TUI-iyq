# 快捷键

> **English**：[KEYBINDINGS.md](KEYBINDINGS.md)

本文是 TUI 识别的全部键盘快捷键**唯一权威目录**。按**上下文**分组——即焦点或模态。列在「Composer」下的绑定仅在合成器聚焦时生效；「Transcript」下仅在转录区聚焦时生效，以此类推。

绑定**尚不支持**用户自定义——未来版本跟踪（#436、#437）。本文是未来配置文件覆盖所命名的契约。

## 全局（任意上下文）

| 组合键 | 动作 |
|--------|------|
| `F1` 或 `Ctrl-?` | 切换帮助浮层 |
| `Ctrl-K` | 打开命令面板（斜杠命令查找） |
| `Ctrl-C` | 取消当前回合 / 关闭模态 / 先确认再退出 |
| `Ctrl-D` | 退出（仅当合成器为空） |
| `Tab` | 循环 TUI 模式：Plan → Agent → YOLO → Plan |
| `Shift-Tab` | 循环推理强度：off → high → max → off |
| `Ctrl-R` | 打开恢复会话选择器 |
| `Ctrl-L` | 刷新 / 清屏 |
| `Ctrl-Shift-E` / `Cmd-Shift-E` | 切换文件树侧栏 |
| `Esc` | 关闭最顶层模态 · 取消斜杠菜单 · 关闭 toast |

## Composer（待发送消息编辑）

| 组合键 | 动作 |
|--------|------|
| `Enter` | 发送消息（或执行斜杠命令） |
| `Alt-Enter` / `Ctrl-J` | 换行不发送 |
| `Ctrl-U` | 删到行首 |
| `Ctrl-W` | 删除前一个词 |
| `Ctrl-A` / `Home` | 移到行首 |
| `Ctrl-E` / `End` | 移到行尾 |
| `Ctrl-←` / `Alt-←` | 后退一个词 |
| `Ctrl-→` / `Alt-→` | 前进一个词 |
| `Ctrl-V` / `Cmd-V` | 从剪贴板粘贴（亦处理 bracketed paste） |
| `Ctrl-Y` | 从 kill buffer 粘贴（yank） |
| `↑` / `↓` | 循环合成器历史（亦用于弹窗/附件项选择） |
| `Ctrl-P` / `Ctrl-N` | 循环合成器历史（替代） |
| `Ctrl-S` | 暂存当前草稿（`/stash list`、`/stash pop` 恢复） |
| `Alt-R` | 搜索提示历史（再按 Alt-R 退出） |
| `Tab` | 斜杠命令 / `@` 提及补全（感知弹窗） |
| `Ctrl-O` | 用外部编辑器打开合成器草稿 |

### `@` 提及

输入 `@<部分路径>` 打开文件提及弹窗。`↑`/`↓` 循环条目，`Tab` 或 `Enter` 接受。`Esc` 关闭弹窗。自 v0.8.10（#441）起，补全按提及 frecency 重排——常提且最近用的文件更靠前。

### `#` 快速添加（记忆）

当 `[memory] enabled = true` 时，输入 `# foo` 后按 `Enter` 会将 `foo` 作为带时间戳的条目追加到记忆文件，**不**发送回合。见 `docs/MEMORY.md`（[简体](MEMORY.zh-CN.md)）。

## Transcript（转录区聚焦时）

| 组合键 | 动作 |
|--------|------|
| `↑` / `↓` / `j` / `k` | 滚动一行（v0.8.13+：合成器为空时裸方向键也可滚动） |
| `PgUp` / `PgDn` | 滚动一页 |
| `Home` / `g` | 跳到顶部 |
| `End` / `G` | 跳到底部 |
| `Esc` | 焦点回到合成器 |
| `y` | 将选中区域复制到剪贴板 |
| `v` | 开始/扩展可视选择 |
| `o` | 打开光标下 URL（需终端支持 OSC 8） |

## Sidebar（侧栏聚焦时）

| 组合键 | 动作 |
|--------|------|
| `↑` / `↓` / `j` / `k` | 移动选择 |
| `Enter` | 激活选中项（打开 / 聚焦 / 取消） |
| `Tab` | 下一侧栏面板（Files → Tasks → Agents → Todos） |
| `Esc` | 焦点回到合成器 |

## 斜杠命令面板（`Ctrl-K` 或输入 `/` 后）

| 组合键 | 动作 |
|--------|------|
| `↑` / `↓` | 移动选择 |
| `Enter` / `Tab` | 运行 / 补全高亮命令 |
| `Esc` | 关闭面板 |

## 审批模态（工具请求审批时）

| 组合键 | 动作 |
|--------|------|
| `y` / `Y` | 批准一次 |
| `a` / `A` | 全部批准（后续同类型自动批准） |
| `n` / `N` / `Esc` | 拒绝 |
| `e` | 运行前编辑已批准输入 |

## 引导（首次运行流程）

| 组合键 | 动作 |
|--------|------|
| `Enter` | 下一步（Welcome → Language → API → …） |
| `Esc` | 返回上一屏 |
| `1`–`5` | 选择语言（语言步骤） |
| `y` / `Y` | 信任工作区（信任步骤） |
| `n` / `N` | 跳过信任提示 |

## v0.8.13 审计说明

- **Ctrl-S 是暂存，不是历史搜索。** 本版已修正 —— `Alt-R` 为历史搜索。
- **已移除不存在的 `Alt+Up`。** README 曾列「编辑最后排队消息」，但按键分发中从未实现。
- **合成器为空时裸 Up/Down 滚动转录（v0.8.13）。** 此前 `should_scroll_with_arrows` 恒为 false，裸方向键总在合成器历史中导航；虚拟终端用户（Ghostty、Codex、Kitty 协议等）尤其受影响。
- **可配置键位（#436）与 `tui.toml`（#437）仍推迟。** `TuiPrefs` 与加载器在 `settings.rs` 存在，启动时尚未接线。
- **未发现其它失效绑定。** 上文其余组合均在 `crates/tui/src/tui/ui.rs`（按键分发）或 `crates/tui/src/tui/app.rs`（模式与状态）有对应处理。
